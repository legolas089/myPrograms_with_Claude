# Half-Car Suspension Simulator

4-DOF 하프카 서스펜션 시뮬레이터. 전/후 현가 입력에 대한 차체 수직 변위와 피치 응답을 실시간으로 시각화합니다.

## 구조

```
half_car_sim/
├── index.html              # 메인 페이지
├── css/style.css           # 다크 테마 스타일
├── js/
│   ├── main.js             # UI, 이벤트, 재생 제어
│   ├── simulation.js       # 4-DOF RK4 솔버, 주파수 응답(4x4 복소수 해석), Comfort 지표
│   ├── road.js             # 노면 프로파일 + 전/후축 시간지연
│   ├── animation.js        # 2D 하프카 스키매틱 애니메이션 (피치 회전 포함)
│   └── graphs.js           # 변위/피치각/현가스트로크/주파수응답 그래프
│
├── Half-Car Sim.bat        # 웹 서버 실행 (port 3003)
└── README.md
```

## 운동방정식

```
F_f = K₁(y₃ - b₁φ₃ - y₁) + C₁(ẏ₃ - b₁φ̇₃ - ẏ₁)
F_r = K₂(y₃ + b₂φ₃ - y₂) + C₂(ẏ₃ + b₂φ̇₃ - ẏ₂)

m₁ÿ₁   = F_f - kₜ₁(y₁ - u₁)
m₂ÿ₂   = F_r - kₜ₂(y₂ - u₂)
m₃ÿ₃   = -F_f - F_r
I₃φ̈₃   = b₁F_f - b₂F_r
```

- `y₁`, `y₂` : 전방/후방 비현가 질량 변위
- `y₃` : 스프렁매스 무게중심(CG) 수직 변위
- `φ₃` : 피치 각도 (positive = nose down)
- `m₃`, `I₃` : 스프렁매스 질량 / 피치 관성모멘트
- `b₁`, `b₂` : CG에서 전축/후축까지 거리
- `K₁`, `C₁`, `kₜ₁` : 전방 스프링/댐퍼/타이어 강성
- `K₂`, `C₂`, `kₜ₂` : 후방 스프링/댐퍼/타이어 강성
- `u₁`, `u₂` : 전방/후방 노면 입력 (시간지연: u₂(t) = u₁(t - L/v), L = b₁ + b₂)

## 사용법

1. `Half-Car Sim.bat` 실행 (http://localhost:3003)
2. 파라미터 조정 → 노면 선택 → [시작]
3. Set A / Set B 비교 모드 가능

## 기능

| 기능 | 설명 |
|------|------|
| 파라미터 슬라이더 | m₃, I₃, b₁, b₂, m₁, K₁, C₁, kₜ₁, m₂, K₂, C₂, kₜ₂ (12개) |
| 파생 지표 | 전/후 댐핑비(ζ), 고유진동수(fₙ), 피치 고유진동수 |
| 노면 프로파일 | Speed Bump, Step, Sine Wave, Random |
| 축간거리 시간지연 | 전륜 → 후륜 (b₁+b₂)/v 후 통과 |
| 비교 모드 | Set A vs Set B 동시 애니메이션 |
| 하프카 스키매틱 | 바디(피치 기울기) + 전/후 서스펜션 + 비현가매스 + 타이어 |
| 변위 그래프 | y₃(CG), y₁(전방), y₂(후방), 노면 시간 이력 |
| 피치각 그래프 | φ₃(t) (deg 단위) |
| 현가 스트로크 그래프 | 전방/후방 서스펜션 상대변위 (mm) |
| 주파수 응답 | CG 가속도, 피치 가속도, 전/후 타이어 변형 (log-log) |
| Ride Comfort | CG/피치 가속도 RMS, 전/후 현가변위, 전/후 타이어변형 + 수식 툴팁 |
| 프리셋 | 승용차 / SUV / 스포츠카 |
