# 3D 3-DOF Robot Arm Simulator

수직다관절 3-DOF 로봇 암 시뮬레이터. 위치 **A(Pick)** 에서 **B(Place)** 로 끝단을 옮기는 다양한 3차원 경로를 자동으로 탐색·비교합니다.

J4(툴 회전) 없이 위치 (x, y, z) 만 결정하는 단순화 모델 — 응력 해석 전 단계의 기하학적 경로 탐색이 목적입니다.

---

## 실행 방법

1. `RobotArmSimulator3D.exe` 를 더블클릭합니다.
2. 검정색 콘솔 창이 하나 뜨고, 잠시 후 기본 브라우저에서 자동으로 페이지가 열립니다.
3. 종료하려면 콘솔 창을 닫으면 됩니다.

> **중요**: 콘솔 창은 내장 웹서버입니다. 시뮬레이터를 쓰는 동안 닫지 마세요.

---

## 로봇 정의

| 관절 | 변수 | 종류 | 회전축 | 기능 |
|:---:|:---:|:---:|:---|:---|
| **J1** | θ1 | R | Z₀ (수직) | 베이스 회전 |
| **J2** | θ2 | R | 수평축 (J1 따라 회전) | 상완 피치 (들어올림) |
| **J3** | θ3 | R | J2 평행 | 전완 피치 (굽힘) |

| 링크 | 길이 | 역할 |
|:---:|:---:|:---|
| L₀ | h0 | 고정 베이스 |
| L₁ | L1 | 상완 |
| L₂ | L2 | 전완 (끝단 P) |
| L_p | L1 | 평행링크 (시각용, 독립 DOF 추가 없음) |

### 정기구학

```
r = L1·cos(θ2) + L2·cos(θ2 + θ3)
x = r · cos(θ1)
y = r · sin(θ1)
z = h0 + L1·sin(θ2) + L2·sin(θ2 + θ3)
```

### 작업공간 (torus)

| 항목 | 식 |
|:---|:---|
| 최대 반경 | `r_max = L1 + L2` |
| 최소 반경 | `r_min = |L1 − L2|` |
| 최대 높이 | `z_max = h0 + (L1 + L2)` |
| 최소 높이 | `z_min = h0 − (L1 + L2)` |

## 경로 생성 (6 전략)

`Explore Paths` 버튼을 누르면 6 가지 전략으로 A→B 경로를 동시에 생성합니다. 각 경로는 80개의 웨이포인트로 샘플링됩니다.

| # | 전략 | 산업 명령어 | 원리 |
|---|------|------------|------|
| 1 | **Joint Linear** | MoveJ / PTP | 관절공간에서 θ1·θ2·θ3 를 선형 보간. IK 불필요. |
| 2 | **Cartesian Linear** | MoveL / LIN | (x,y,z) 직선 보간 + 매 스텝 IK. 직선 경로. |
| 3 | **Via-Point Spline** | Zone / APO / CNT | 랜덤 경유점을 통과하는 3D 2차 Bezier. |
| 4 | **Elbow Switch** | confdata / STATUS | Elbow-Up ↔ Down 시그모이드 블렌드. |
| 5 | **Cubic Polynomial** | 내부 보간 엔진 | `q(t)=a₀+a₁t+a₂t²+a₃t³`, 경계 속도 변화. |
| 6 | **Circular Arc** | MoveC / CIRC 변형 | 관절공간 직선에 직교 offset. |

### 비용 메트릭

```
cost = Σ|Δq| · 1.0 + Σ|ΔP_cart| · 1.0 + max|Δq| · 5.0 + Σ|Δ²q| · 3.0
```

(가중치는 2D 버전과 동일하나, 카테시안 항만 [m] 단위에 맞춰 1.0 으로 조정)

## 사용 방법

1. **암 파라미터** 설정: `h0`, `L1`, `L2` 슬라이더 [m]
2. **(선택) 관절 제한**: `Enable Joint Limits` 체크 후 θ1·θ2·θ3 min/max 설정 [°]
3. **위치 A·B**: 슬라이더로 `(x, y, z)` 또는 3D 뷰의 마커를 마우스 드래그 (XY 평면 상에서 이동, z 는 슬라이더로)
4. **Explore Paths**: 6 전략 결과를 비용 순으로 정렬해 표시
5. **경로 선택**: 우측 목록 클릭 → 3D 뷰·그래프에 강조
6. **Animation**: `Play` → 로봇팔이 선택 경로를 따라 움직이고 박스가 A→B 운반
7. **카메라**: `Iso / Top / Front / Side` 프리셋 + 마우스 orbit/zoom/pan
8. **CSV 내보내기**: `Export Selected Path as CSV` → `t1, t2, t3, x, y, z` 컬럼

## 시각화

| 요소 | 의미 |
|---|---|
| 회색 원통 | 베이스 (L0) |
| **청록 막대** | 상완 (L1) |
| **주황 막대** | 전완 (L2), 끝단 P |
| 회색 가는 막대 | 평행링크 L_p (시각용) |
| 노란 구체 | 회전 관절 (J2, J3) |
| 빨간 구체 | 끝단 (P) |
| 노란 박스 | 운반 페이로드 (애니메이션 중) |
| 파란 와이어 토러스 | 도달 가능 작업공간 |
| 빨강/초록 구체 | 위치 A (Pick) / B (Place) |
| 색상 라인 | 6 전략으로 생성된 경로 (선택 경로 두꺼움) |

## 주의사항

1. **독립 DOF는 3** — 평행링크는 시각용이며 자유도를 추가하지 않습니다.
2. **자세(orientation) 미제어** — 끝단의 위치 (x, y, z) 만 결정합니다.
3. **특이점**:
   - `r → 0` (팔 완전 접힘): J1 회전이 무의미
   - `r = r_max` 경계: 자코비안 rank deficiency
4. **IK 해 2개**: Elbow-Up / Elbow-Down 중 자동 선택. Elbow Switch 전략은 두 해를 부드럽게 전환합니다.
